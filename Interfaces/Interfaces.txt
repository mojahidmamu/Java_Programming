1. Interfaces > what?
    => Interfaces in a contact in java.
    => অন্য কোনো class এই Interface-কে implement করলে, 
        সেই class-কে Interface-এর সব method define/implement করতে হয়।

2. in easy word: 
    i. Interface = Rule book

3. Important point: 
    i. Interface-এর সব method by default abstract (মানে body ছাড়া)।
    ii. Interface-এর সব variable by default public, static, final হয়।
    iii. Class → Interface implement করে implements keyword দিয়ে।
    iv. এক class একাধিক interface implement করতে পারে।
    v. Interface অন্য Interface-কে extend করতে পারে।
    vi. Interface-এর object তৈরি করা যায় না, কিন্তু reference করা যায়:

4. Why we use (Interface):
    i. Abstraction (implementation লুকানো)
    ii. Multiple inheritance (class দিয়ে সম্ভব নয়, কিন্তু interface দিয়ে হয়)
    iii. Loose coupling (class গুলো একে অপরের উপর কম নির্ভরশীল হয়)
    iv. Code standardization (সবার জন্য একরকম rule তৈরি করা)
