3. Important point: 
    i. Interface-এর সব method by default abstract (মানে body ছাড়া)।
    ii. Interface-এর সব variable by default public, static, final হয়।
    iii. Class → Interface implement করে implements keyword দিয়ে।
    iv. এক class একাধিক interface implement করতে পারে।
    v. Interface অন্য Interface-কে extend করতে পারে।
